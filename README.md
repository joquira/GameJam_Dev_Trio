# Atras da Porta

Projeto da AC1 Game Jam em Godot 4.

## Como jogar

- Selecione `Fase 1 - Ruina` no hub inicial para comecar.
- Use `A/D` ou as setas para mover.
- Use `Espaco` ou `W` para pular.
- Segure `Shift` para correr.
- Pressione `Esc` para pausar a fase.
- Pegue os 2 baus, use o power-up de pulo alto e entre na porta final.
