# Atras da Porta

Projeto da AC1 Game Jam em Godot 4.

Tema: What's behind the door  
Genero: plataforma 2D

## Como jogar

- `A`/seta esquerda: andar para a esquerda
- `D`/seta direita: andar para a direita
- `Shift`: correr
- `Espaco` ou `W`: pular
O objetivo e atravessar a fase, coletar 2 baus flutuantes e encostar na porta final.

## Estrutura

- `scenes/`: cena principal do jogo
- `scripts/`: scripts de player, fase e inimigos
- `assets/`: icone, sprites do personagem e tiles do cenario
- `docs/`: GDD da atividade
